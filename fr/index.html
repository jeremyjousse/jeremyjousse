Using Travis to deploy Middleman on Github pages
I recently created a blog scaffold for a friend of my using Middleman. Using Github pages in addition to Middleman allows my friend to host his blog without any charge. This is great but my friend is not a geek, and he does not hold back all the Middleman boring commands to commit, generate, and deploy his blog. I finaly found prose.io a content editor for GitHub designed for managing websites. With a little my time of configuring prose.io, my friend has finally a GUI editor to handle his Middleman blog. The last challenge was to automate the build and deployment of his blog after each commit.
